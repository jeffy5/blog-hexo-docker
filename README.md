# blog-hexo-docker

> 构建个人 hexo 博客镜像

## 插件

- [hexo-admin](https://github.com/jaredly/hexo-admin)

### Hexo Admin 配置后台密码

在 `config/_config.yml` 配置后台登录密码，密码生成需要通过

```yaml
# hexo-admin authentification
admin:
  username: xxx
  password_hash: xxx
  secret: xxx
```
